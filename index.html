VietQR tự động tạo QR từ thông tin bên dưới — khách quét là thanh toán ngay, không cần cài app riêng.

📊 Đồng bộ đơn hàng vào Google Sheet

Mỗi khi có đơn mới, hệ thống tự động ghi vào Google Sheet. Bạn xem được đơn hàng từ điện thoại / máy tính bất cứ đâu qua app Google Sheets.

Cách setup (làm 1 lần, ~3 phút):
1. Tạo 1 Google Sheet mới tại sheets.new
2. Vào Tiện ích mở rộng → Apps Script
3. Xóa code cũ, dán code dưới đây vào, đổi tên file thành "Code.gs"
4. Bấm Triển khai → Triển khai mới → Loại: Ứng dụng web
5. Chọn "Thực thi với tư cách: Tôi" và "Ai có quyền truy cập: Bất kỳ ai"
6. Copy URL Web App, dán vào ô bên dưới và bấm Lưu
📋 Xem code Apps Script (click để mở)
